Class: ObjectUtil

ObjectUtil

new ObjectUtil(objects)

Creates a new form of Object with more utilities than a standard Object.
Parameters:
Name Type Argument Description
objects Object <repeatable>
 Any Objects you would like to convert into an ObjectUtil. Earlier Objects take priority.
Version:
  • 1.1.0
Source:
Utils/ObjectUtil.js
Example
const obj = new ObjectUtil({ a: 1, b: 2 });
obj.c = 3;
console.log(obj.a, obj.b, obj.c);

Methods

assign(sources)

Assigns the entries of other Objects/ObjectUtils to the ObjectUtil
Parameters:
Name Type Argument Description
sources Object | ObjectUtil <repeatable>
 The Objects/ObjectUtils you would like to add.
Source:
Utils/ObjectUtil.js
Example
// expected output: [ [ 'a', 1 ], [ 'b', 2 ], ['c', 3 ] ]
const obj = new ObjectUtil();
obj.assign({ a: 1 }, { b: 2, c: 3});
console.log(obj.entries());
