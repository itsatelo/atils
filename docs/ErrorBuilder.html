Class: ErrorBuilder

ErrorBuilder

new ErrorBuilder(name, message [, options])

- A simple Error Builder with a customized catch method.
Parameters:
Name Type Argument Description
name string The name of the Error.
message string The message of the Error.
options Object <optional>
 Options for the Error.
Properties
Name Type Argument Description
date Boolean <optional>
 Whether to display the current date (in a MM/DD/YYYY HH:MM:SS format) within the Error.
stackTrace Boolean <optional>
 Whether to display the Stack Trace within the Error.
logStackTraces Boolean <optional>
 Whether to log Stack Traces into a folder (folder is made by atils).
autocatch Boolean <optional>
 Whether to automatically catch this Error when thrown.
Source:
  • atils@1.2.x/Utilities/Error.js
Example
const { ErrorBuilder } = require('atils');
const ExampleError = new ErrorBuilder('Example Error', 'This is an example.');
ExampleError.throw();

Methods

catch( [enable])

Enables/Disables "Catch Mode", a simple Catching Mechanism for the Error Builder.
Parameters:
Name Type Argument Description
enable Boolean <optional>
 Whether Catch Mode should be enabled or disabled.

  • atils@1.2.x/Utilities/Error.js
