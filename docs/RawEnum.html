Class: RawEnum

RawEnum

new RawEnum(resolverName, rawEnumInfo)

- A Raw Enum that utilizes it's own Resolver.
Parameters:
Name Type Description
resolverName string The Resolver Name for this Raw Enum instance.
rawEnumInfo Object The information for the Raw Enum instance.
Properties
Name Type Argument Description
YOUR_RESOLVER_NAME Object <optional>
 Creates a new Resolver with the items within. Replace "YOUR_RESOLVER_NAME" with whatever you want.
Properties
Name Type Argument Description
YOUR_KEY_NAME * <optional>
 Creates a new Key Value Pair within this Resolver. Replace "YOUR_RESOLVER_NAME" and "YOUR_KEY_NAME" with whatever you want.
YOUR_SECOND_RESOLVER Object <optional>
 Creates a new Resolver with the items within. Replace "YOUR_SECOND_RESOLVER" with whatever you want.
Properties
Name Type Argument Description
YOUR_SECOND_KEY_NAME * <optional>
 Creates a new Key Value Pair within this Resolver. Replace "YOUR_SECOND_RESOLVER" and "YOUR_SECOND_KEY_NAME" with whatever you want.
Deprecated:
  • - Use Enums, not Raw Enums.

Example
const { RawEnum } = require('atils');
const exampleEnum = new RawEnum('testingEnum', { text: "hello world" });
